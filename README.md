## 📖 درباره پروژه

این ریپازیتوری سورس‌کد وب‌سایت پورتفولیوی شخصی من رو در خودش داره. سایت با هدف معرفی مهارت‌ها و توانایی‌های من در چهار حوزه اصلی طراحی شده:

- 💻 **برنامه‌نویسی C#** (اپلیکیشن‌های کنسولی و ویندوزی)
- 🎥 **ویدئوگرافی و تدوین**
- 🎨 **طراحی وب** (Web Design)
- 🔒 **امنیت وب** (Web Security)

طراحی سایت با تم روشن (Light) و انیمیشن‌های اختصاصی ساخته شده تا تجربه‌ی کاربری روان و جذابی ارائه بده.

## ⚙️ نحوه اجرا روی سیستم شخصی (Local)

اگه می‌خوای سایت رو روی سیستم خودت ببینی و تغییرش بدی:

1. ریپازیتوری رو کلون کن:
   ```bash
   git clone https://github.com/roetod/tahaeloney.github.io.git
   ```
2. وارد پوشه پروژه شو:
   ```bash
   cd tahaeloney.github.io
   ```
3. فایل `index.html` رو با مرورگر باز کن — همین! نیازی به نصب چیز خاصی نیست چون پروژه کاملاً استاتیک (Static) هست.
